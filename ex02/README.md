## 超高機能電卓（ex02/calc.py）
### 追加機能
- 四則演算：＋以外䛾四則演算をする
- AC：entryに入力されている数字，数式䛾文字列全体をdeleteする
- %ボタン:あまりつき割り算ができ、あまりが出力される
- テーマ:計算機のデザインを全体的に黒くし見やすくした
- 小数点:小数点付き計算が行えるようにした
- 記号の連続:符号が連続して打てないようにし、計算仮定でエラーが出ないようにした。
